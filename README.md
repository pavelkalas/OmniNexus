# OmniNexus

OmniNexus je Forge mod pro Minecraft 1.12.2, který poskytuje základní systém správy hráčských účtů na serveru.

Mod je primárně určen pro soukromý herní server s kamarády, ale kód je veřejně dostupný, takže si ho může kdokoli upravit, rozšířit nebo použít jako základ pro vlastní řešení.

---

## 🔐 Funkce (zatím implementováno)

### Login systém
- `/login <heslo>` – přihlášení hráče
- `/register <heslo>` – registrace hráče
- `/changepw <heslo>` – změna hesla

## 🎯 Účel projektu

OmniNexus byl vytvořen pro soukromý Minecraft server mezi kamarády.

Projekt ale zůstává otevřený, takže kdokoli ho může:
- použít
- upravit
- rozšířit o vlastní funkce
- integrovat do jiných serverových modů

---

## ⚠️ Poznámka

Tento mod není zaměřený na produkční/veřejné servery s vysokým zatížením.
Použití je na vlastní riziko.

---

## 🛠️ Kompatibilita

- Minecraft: 1.12.2  
- Forge: 14.23.x (doporučeno)
- Java: 8+
